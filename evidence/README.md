# SignalPilot ‚Äî Evidence Campaign

This directory contains HTML reports, JSON analysis dumps, console captures, and timing
files generated by running SignalPilot against 12 live failure scenarios on K3s.

Each file group per scenario:

| File | Description |
|------|-------------|
| `<name>_report.html` | Self-contained HTML report (for blog screenshots) |
| `<name>_analysis.json` | Full Analysis JSON dump |
| `<name>_console.txt` | Rich CLI table + narrative captured as text |
| `<name>_timing.txt` | Detection time in seconds |
